# Simulated Landing Page

## Page Title
Account Verification Portal

## Purpose
This page simulates a fake login portal used in a phishing awareness exercise. It is intended to measure whether users would attempt to enter credentials after clicking a suspicious link.

## Observed User Action
The simulation tracks whether the user:
- Reached the page
- Attempted to interact with the form
- Submitted credentials

## Note
This page is part of a controlled lab exercise and is not connected to any real authentication system.
