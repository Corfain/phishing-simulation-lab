# Phishing Campaign Simulation Lab

This project demonstrates a controlled phishing awareness simulation conducted in a lab environment for security education purposes. The goal is to analyze user behavior, measure phishing susceptibility, and develop remediation strategies.

---

## Objectives
- Simulate a phishing campaign in a controlled environment
- Measure user interaction with phishing content
- Analyze awareness weaknesses
- Create a remediation plan to reduce future risk

---

## Scope
This project is designed only for lab-based security awareness testing and defensive education. It does not target real users or production systems.
